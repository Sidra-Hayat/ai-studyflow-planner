# 📚 AI StudyFlow Planner

A smart AI-powered study planning web application that helps students manage subjects, generate optimized study schedules, track progress, and improve productivity.

## 🚀 Live Demo
https://sidra-hayat.github.io/ai-studyflow-planner/

---

## ✨ Features
- 📅 AI-based study plan generator
- 📊 Progress tracking system
- 🧠 Smart priority algorithm (difficulty + exam date)
- 📈 Weekly performance analytics (Chart.js)
- 🗓️ Calendar-based study visualization
- 🔔 Daily schedule breakdown
- 🤖 AI Study Advisor (Claude API integration)
- 💾 Local storage data persistence

---

## 🛠️ Tech Stack
- HTML5
- CSS3 (Modern UI)
- JavaScript (ES6+)
- Chart.js
- LocalStorage API
- Claude AI API
